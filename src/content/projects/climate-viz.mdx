An exploration of climate data through interactive visualization.

## Overview

This project visualizes historical climate data to make patterns more accessible and understandable.

## Features

- Interactive time series exploration
- Geographic data mapping
- Comparative analysis tools

## Built With

This project uses D3.js and Scrollama for scroll-driven storytelling. The data comes from publicly available climate datasets.
